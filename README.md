
# 🚀 Question Difficulty Classifier

This project uses NLP and Machine Learning to classify questions into:
- Easy
- Medium
- Hard

## 🔧 Technologies Used
- Python
- Scikit-learn
- TF-IDF
- LinearSVC
- T5 Transformer
- Flask

## 📊 Features
- Text preprocessing
- ML model training
- Deep learning with T5
- REST API for predictions

## 📈 Results
Achieved ~75-80% accuracy using optimized models.

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
